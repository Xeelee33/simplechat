Current release version: **0.250.001**

Document processing gained Standard, Enhanced, and Auto extraction, consistent metadata extraction, PDF reprocessing, media citation badges, legacy Office support, and native Visio ingestion with previews.

## User Side

Workspace users can inspect extraction badges, metadata status, enhanced citation badges, PDF reprocessing actions, and Visio citation previews.

## Admin Side

Admins configure Document Intelligence extraction behavior, Auto sample pages, citation behavior, and supported processing features from Search & Extract and Citations settings.

1. Open **Admin Settings > Search & Extract** and review extraction modes.
2. Upload a PDF, image, media file, legacy Office file, or `.vsdx` diagram.
3. Inspect workspace details and enhanced citations.
