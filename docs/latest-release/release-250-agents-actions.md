Current release version: **0.250.001**

Agents gained Assigned Knowledge, catalog customization, voice-assisted instruction drafting, Databricks, Tableau, Microsoft Graph mail, MCP actions, reusable identities, and better action governance.

## User Side

Users can find agents with clearer catalog metadata, use assigned-knowledge agents for governed sources, and work with actions that use configured reusable identities.

## Admin Side

Admins manage global identities, action-type availability, model endpoint access, agent catalog customization, and advanced actions such as Databricks, Tableau, Graph mail, and MCP.

3. Open workspace or global identities when an action needs reusable credentials.
