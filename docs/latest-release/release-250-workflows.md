Current release version: **0.250.001**

Workflows are a major new automation surface. They can run prompts over selected documents, process each document separately, monitor File Sync changes, resume failed batches, and create Word or PowerPoint outputs when those actions are enabled.

## User Side

Users can create or run personal and group workflows for repeatable document analysis, File Sync refreshes, per-document runs, and generated Office outputs.

## Admin Side

Availability may depend on tenant settings, governance policies, workspace scope, action configuration, or admin-managed identities. If you do not see this capability, contact your SimpleChat admin.

## Why It Matters

This matters because repeatable document work can move from one-off chat prompts into reusable personal or shared group automation.

## How to Try It

1. Open Personal Workspace > Workflows when personal workflows are enabled for your account.
2. Open Group Workspaces to use shared group workflows when your group has access.
3. Use history and activity views to inspect completed, running, or failed workflow runs.
