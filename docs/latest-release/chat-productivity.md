Current release version: **0.241.183**

Chat Productivity groups the everyday improvements that make repeated chat work faster and more comfortable, including Analyze and Compare document actions from the Workspaces tool.

## User Side

Users work from the chat composer: attach files or URLs, switch models, choose prompts and agents, ground messages in workspace sources, paste screenshots, choose Search, Analyze, or Compare for selected documents, and continue from generated charts or artifacts.

## Admin Side

Admins enable the underlying chat, upload, model endpoint, Source Review, and workspace capabilities that determine which composer tools are visible in a given deployment.

## Why It Matters

Frequent chat tasks should feel lighter, especially when users have many conversations, pasted images, long-running answers, or reusable generated outputs.

## How to Try It

1. Paste copied screenshots or browser-exposed files directly into the chat input when uploads are enabled.
2. Use Analyze when selected documents need full review instead of search snippets.
3. Use Compare when one source document should be checked against one or more target documents.
4. Use conversation search and **Load More** behavior when browsing larger conversation histories.
5. Try suggested prompt buttons, message masking, inline charts, and generated Markdown previews when those controls appear in a response.

## Notes

- Clipboard uploads depend on browser support and enabled upload settings.
- Analyze and Compare depend on workspace document access and selected document scope.
- Paged conversation loading keeps startup faster for users with many conversations.
- Charts and artifact previews are designed to keep analysis results easier to inspect in place.
