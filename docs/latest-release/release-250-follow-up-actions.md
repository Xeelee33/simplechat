Current release version: **0.250.001**

When a response includes supported next-step suggestions, SimpleChat can render them as clickable prompt actions below the assistant message. Users can continue a workflow without copying and pasting suggested text.

## User Side

Assistant responses can now show suggested next-step buttons that stage the prompt and start a cancelable send countdown.

## Admin Side

Availability may depend on tenant settings, governance policies, workspace scope, action configuration, or admin-managed identities. If you do not see this capability, contact your SimpleChat admin.

## Why It Matters

This matters because useful assistant suggestions become one-click follow-up actions while users stay in control before sending.

## How to Try It

1. Click a suggested follow-up action when it matches what you want to do next.
2. Use the countdown window to cancel before the prompt is sent.
3. Edit the staged prompt if you want to customize the next step.
