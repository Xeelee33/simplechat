Current release version: **0.250.001**

Generated artifact cards make structured outputs easier to reuse. Markdown can be viewed in Chat, generated Office files can support workflow outputs, and reusable artifacts can become workspace documents with approval where needed.

## User Side

Agents and workflows can now create reusable Markdown, Word, and PowerPoint outputs that users can inspect, download, or promote into workspaces.

## Admin Side

Availability may depend on tenant settings, governance policies, workspace scope, action configuration, or admin-managed identities. If you do not see this capability, contact your SimpleChat admin.

## Why It Matters

This matters because important results should become durable files when users need reports, decks, summaries, or workspace knowledge.

## How to Try It

1. Use generated artifact cards to view or download outputs from Chat.
2. Use Add to Workspace when a generated output should become searchable knowledge.
3. Use workflows when repeatable document analysis should produce Word or PowerPoint outputs.
