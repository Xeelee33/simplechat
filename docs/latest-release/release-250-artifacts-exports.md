Current release version: **0.250.001**

Generated Markdown viewing, inline charts, tabular exports, related-document evidence, generated Office files, per-message export, and conversation PDF or summary export make outputs easier to reuse.

## User Side

Users can inspect generated Markdown, view inline charts, download or promote generated artifacts, export individual messages, and export conversations with summaries or PDF output.

## Admin Side

Admins govern upload capabilities, generated Office tool availability, workspace promotion approval behavior, and export-related storage and retention paths.

## Why It Matters

Important results should not be trapped in a single chat bubble when users need a document, chart, spreadsheet-style output, or reusable workspace artifact.

## How to Try It

1. Ask Chat for a chart, Markdown artifact, tabular export, or document output.
2. Use View, Download, Add to Workspace, or message-level export actions.
3. Export a conversation as Markdown, JSON, or PDF when a full handoff is needed.

## Notes

- Large tabular outputs can stay conversation-scoped as downloadable artifacts instead of flooding the visible answer.
