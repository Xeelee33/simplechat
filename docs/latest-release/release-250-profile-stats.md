Current release version: **0.250.001**

Users can review activity windows, export stats, manage settings, inspect group and public workspace membership, and tune navigation, tutorial, memory, speech, and voice preferences from Profile.

## User Side

Profile now brings together stats, groups, public workspaces, feedback, safety items, preferences, and CSV exports in a clearer experience.

## Admin Side

Availability may depend on tenant settings, governance policies, workspace scope, action configuration, or admin-managed identities. If you do not see this capability, contact your SimpleChat admin.

## Why It Matters

This matters because users can understand their own activity and manage everyday preferences without needing an admin to change global settings.

## How to Try It

1. Open Profile > Stats to review 7-day, 30-day, 90-day, or custom reporting windows.
2. Use Profile tabs to review groups, public workspaces, feedback, and safety items.
3. Use Profile > Settings to control navigation state, tutorial visibility, memories, speech, and voice preferences.
