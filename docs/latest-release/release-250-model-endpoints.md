Current release version: **0.250.001**

Model endpoint setup guidance, global and workspace endpoint management, provider-aware Foundry routing, Claude-capable deployments, and model icon avatars make model choices clearer across Chat, agents, workflows, and generated summaries.

## User Side

Users choose from the model picker and see model-only assistant responses with clearer visual identity when endpoint icons or uploaded model images are configured. Agent responses keep agent avatar priority.

## Admin Side

Admins and workspace managers configure Azure OpenAI, Foundry, and New Foundry endpoints with setup guidance, model discovery, test operations, RBAC notes, and provider metadata.

## Why It Matters

Model selection is more flexible and easier to explain when teams offer multiple providers, endpoints, and deployment families.

## How to Try It

1. Open **Admin Settings > AI Models** and review endpoint setup guidance.
2. Configure or inspect a model endpoint icon or uploaded model image.
3. Open Chat and select an available model from the model picker.
