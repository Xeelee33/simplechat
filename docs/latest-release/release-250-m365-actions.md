Current release version: **0.250.001**

The Microsoft Graph action family can support user mailbox and calendar workflows, including creating drafts, delayed-delivery drafts, sending mail, and working with calendar information when configured by admins.

## User Side

Microsoft Graph actions expand M365 support so approved users can work with mail, drafts, calendar details, and calendar invites from SimpleChat.

## Admin Side

Availability may depend on tenant settings, governance policies, workspace scope, action configuration, or admin-managed identities. If you do not see this capability, contact your SimpleChat admin.

## Why It Matters

This matters because common M365 tasks can become part of an agent-assisted workflow instead of requiring manual copying between apps.

## How to Try It

1. Use an M365-enabled action or agent when you need email or calendar assistance.
2. Review prepared drafts before sending when your environment uses manual draft mode.
3. If M365 actions are unavailable, admins may need to grant scopes or enable the action for your workspace.
