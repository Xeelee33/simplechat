Current release version: **0.250.001**

Cards and folder-card views help users scan files visually, browse by tags, review document details, and open document actions from personal, group, and public workspaces.

## User Side

Workspace documents can now be browsed in list, card, folder, and folder-plus-card views with improved multi-select and action behavior.

## Admin Side

Availability may depend on tenant settings, governance policies, workspace scope, action configuration, or admin-managed identities. If you do not see this capability, contact your SimpleChat admin.

## Why It Matters

This matters because large workspaces are easier to navigate when users can choose the browsing mode that fits the task.

## How to Try It

1. Use List for dense scanning, Cards for visual browsing, Folders for tag-first navigation, and Folders + Cards for both together.
2. Use visible-only select-all and multi-select tools for bulk cleanup or organization.
3. Click cards to open document actions such as Chat, Edit, Select, or management controls.
