Current release version: **0.250.001**

The Tableau action adds a read-only way to explore Tableau Server or Tableau Cloud metadata through an approved SimpleChat action. Access may be limited by admins, workspace configuration, or Tableau credentials.

## User Side

Users with access can ask SimpleChat to discover Tableau projects, workbooks, views, datasources, and workbook details from approved Tableau environments.

## Admin Side

Availability may depend on tenant settings, governance policies, workspace scope, action configuration, or admin-managed identities. If you do not see this capability, contact your SimpleChat admin.

## Why It Matters

This matters because users can find and reason about Tableau assets without manually switching between systems for every lookup.

## How to Try It

1. Use a Tableau-enabled agent or action when you need workbook, view, datasource, or project discovery.
2. If Tableau is not available, ask an admin whether the action is enabled for your workspace or account.
3. Treat Tableau actions as read-only discovery tools unless your admins document additional behavior.
